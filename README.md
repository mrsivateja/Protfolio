# Protfolio

Personal developer portfolio built with React, TypeScript, Vite, and Tailwind CSS. Showcases projects, skills, education, and contact information with a clean, responsive, and animated UI.

## Overview

This repository contains a component-driven portfolio site featuring project highlights, an interactive skills section, education, and a contact form. It uses Framer Motion for subtle animations and a set of reusable UI primitives under `src/components/ui/` to keep layouts consistent and easy to maintain.

## Highlights

- Component-based architecture with reusable UI building blocks
- Animated, accessible, and mobile-first design using Tailwind CSS + Framer Motion
- Pages for Projects, Skills, Education, and Contact
- Easy to customize: add or replace projects in `src/components/portfolio/`

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Run locally

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
